<h1 align="center">Com vida</h1>

<div align="center">

A platform for publicizing emergency cases against covid-19 in Brazil

## ✨ Features

- 🌈 User registration
- 🌍 Search by location
- 🛡 Password recovery
- 🤝 External link for communication on WhatsApp

## 🖥 Environment Support

- Modern browsers



## 🔨 Main packages used

- React
- Styled Components
- Redux
- Redux Saga
- Immer
- Axios
- Formik
- Yup
- Toastify
